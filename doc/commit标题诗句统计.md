# Commit 标题诗句统计

截至第845个commit。之后如有较多更新，有可能再次统计。

本仓库有一个很有辨识度的习惯：大量 commit 标题使用古诗文名句。本文按当前 `master` 分支的 git 历史做一次粗略统计。

统计口径：读取 `git log --format=%s` 的 commit 标题，提取标题开头的中文诗句作为候选；对候选句做简繁归一后，使用中文诗词语料反查作者和作品，并对常见古诗文来源做少量人工补齐。由于部分标题是普通中文说明，部分诗句存在异文、乐府题名、语料缺漏，结果适合作为趋势统计，不宜视为严格文献校勘。

## 结论

李白是这个仓库 commit 标题里最强势的来源，已归类次数达到 117 次，明显高于第二名杜甫的 72 次。白居易位列第三，但作品集中度很高，《琵琶行》和《长恨歌》贡献了大量标题。

按作品看，长诗和长篇古文更容易反复出现。《琵琶行》《长恨歌》《将进酒》《春江花月夜》《离骚》都不只提供一句名句，而是作为一个可持续取材的标题库反复出现。

同一句完全重复使用的次数整体不高，常见重复句多为特别醒目的名句，如《将进酒》中的几句、《蜀相》《茅屋为秋风所破歌》《琵琶行》等。这说明标题习惯更偏向“从同一首长诗里继续取不同句”，而不是简单重复同一句。

总体风格很鲜明：浪漫豪放的李白占主导，杜甫提供厚重叙事和家国感，白居易贡献长篇叙事诗的稳定素材；再辅以王维、苏轼、陆游、李商隐、辛弃疾等，构成了一个相当古典的 commit 标题谱系。
