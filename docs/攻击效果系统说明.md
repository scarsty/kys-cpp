# 攻击效果（AttackEffect）系统说明

本文档介绍 Hades 即时战斗模式中 `AttackEffect` 的各字段含义及各操作类型的工作方式。

---

## 一、AttackEffect 结构体字段说明

### 基础定位与运动

| 字段 | 类型 | 说明 |
|------|------|------|
| `Pos` | `Pointf` | 攻击效果的当前物理坐标 |
| `Velocity` | `Pointf` | 每帧移动向量（速度），决定方向和速率 |
| `Acceleration` | `Pointf` | 每帧加速度，会叠加到 `Velocity` 上 |

每帧更新逻辑：
```
Velocity += Acceleration
Pos += Velocity
```

### 帧计数与生命周期

| 字段 | 类型 | 说明 |
|------|------|------|
| `Frame` | `int` | 当前已经过的帧数，每帧 +1 |
| `TotalFrame` | `int` | 效果的总生命期帧数，`Frame` 超过此值后效果被移除 |
| `TotalEffectFrame` | `int` | 贴图动画的总帧数，用于循环播放：`Frame % TotalEffectFrame` |

- **渲染透明度**：随 `Frame` 增大逐渐变透明，公式为 `alpha = 255 * (TotalFrame * 1.25 - Frame) / (TotalFrame * 1.25)`
- **距离衰减**：伤害乘以 `(1 - 0.3 * Frame / TotalFrame)`，即飞行越远伤害越低（最多衰减 30%）

### 伤害修正

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `Strengthen` | `double` | 1.0 | 伤害乘数。> 1 为增伤，< 1 为减伤 |
| `Weaken` | `int` | 0 | 伤害减免，先减去此值再乘以 `Strengthen` |

最终伤害计算流程：
```
hurt = 基础伤害计算(攻击者, 防御者, 武功)
hurt -= Weaken        // 减去弱化值
hurt *= Strengthen    // 乘以强化倍率
hurt *= 距离衰减      // (1 - 0.3 * Frame / TotalFrame)
hurt *= 角度修正      // 侧面 1.2x，背面 1.5x
hurt *= 操作类型修正  // 见下文各操作类型说明
```

### 追踪与贯穿

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `Track` | `int` | 0 | 是否追踪最近敌人（0=否，1=是） |
| `Through` | `int` | 0 | 是否贯穿（0=击中后停止，1=穿透继续飞行） |
| `NoHurt` | `int` | 0 | 是否无伤害（效果仍渲染但不造成伤害） |

**追踪逻辑**（`Track == 1` 时）：
- 每帧找到最近敌人，计算一个朝向敌人的修正向量（大小为当前速度的 1/20）
- 将修正向量叠加到 `Velocity`，然后重新归一化到原始速率
- 结果是弹道平滑弯曲飞向目标

**非贯穿命中后**（`Through == 0`）：
- 设置 `NoHurt = 1`（不再造成伤害）
- 将 `Frame` 快进到 `max(TotalFrame - 15, Frame)`，使效果快速淡出消失

**贯穿**（`Through == 1`）：
- 击中后继续飞行，可命中多个敌人
- 每个敌人最多被同一个弹道击中一次（通过 `Defender` map 记录）

### 其他字段

| 字段 | 说明 |
|------|------|
| `Attacker` | 攻击发起者（`Role*`） |
| `Defender` | 已被此效果击中的敌人列表（`map<Role*, int>`），防止重复伤害 |
| `UsingMagic` | 使用的武功 |
| `UsingHiddenWeapon` | 使用的暗器（不为空时走暗器伤害公式） |
| `OperationType` | 操作类型编号，决定攻击模式（详见下文） |
| `IsUltimate` | 大招标记，为 1 时会触发额外弹道和增强效果 |
| `FollowRole` | 若设定，效果会跟随指定角色移动 |
| `Path` | 效果贴图资源路径 |

## 二、操作类型（OperationType）详解

操作类型决定了攻击效果的创建方式、弹道行为和命中后的特殊效果。当角色的 `OperationType == -1` 时，系统根据武功的 `AttackAreaType` 自动分配：

| AttackAreaType | 自动分配的 OperationType |
|------|------|
| 0 | 0（近身连击） |
| 1 或 2 | 2（远程弹射） |
| 3 | 1（散射追踪） |

---

### OperationType 0 — 近身连击

**创建规则：**
- 效果直接生成在角色面前（距角色 2 个 TILE_W）
- 连击计数器 `OperationCount` 每次攻击 +1（超时 120 帧或满 3 次重置）
- **普通攻击（第 1、2 次）**：
  - `TotalFrame = 10`（很短，近身快速命中）
  - 无速度，原地生效
- **终结技（第 3 次或大招）**：
  - `TotalFrame = 30`
  - `Strengthen = 2`（双倍伤害）
  - `Track = 1`（追踪敌人）
  - 速度设为武功选择距离的一半，向角色面朝方向发射

**大招增强：**
- 额外生成 1 个追踪弹道（splash）
  - `Strengthen = 0.5`（半伤）
  - `Track = 1`
  - `TotalFrame = 60`
  - 方向偏转 ±30°，速度 3

**命中效果：** 无额外修正，标准击退（强度 2.0，上限 3.0）

---

### OperationType 1 — 散射追踪

**创建规则：**
- 发射 1 个追踪弹道（大招时加倍为 2 个）
- `TotalFrame = 120`（长生命期）
- 方向在角色面朝方向 ±5° 内随机
- `Track = 1`（全部追踪）
- 速度 3
- `Frame` 初始随机 0~10（错开命中时机）

**命中效果：**
- 伤害乘以 **1.5 倍**
- 标准击退（强度 2.0，上限 3.0）

---

### OperationType 2 — 远程弹射

**创建规则：**
- 自动瞄准最近敌人发射（若无敌人则沿面朝方向）
- 速度为 `PROJECTILE_SPEED`（固定弹速）
- `TotalFrame` 由武功射程动态计算：`calcProjectileFrames(射程, TILE_W * 2)`
- 若武功 `AttackAreaType` 为 1 或 2，则主弹道设为 **贯穿**（`Through = 1`）

**范围攻击增强**（AttackAreaType 1 或 2）：
- 额外发射侧向弹道：
  - 普通：2 个，`Strengthen = 0.2`
  - 大招：3 个，`Strengthen = 0.35`
  - 方向在主弹道 ±15° 内随机，速度递减
  - 全部贯穿（`Through = 1`）

**命中效果：**
- 无额外伤害修正
- **低击退**（强度 0.5，上限 1.5），适合远程定点输出

---

### OperationType 3 — 突进冰冻

**创建规则：**
- 角色自身向面朝方向突进（设置 `Velocity`，速度由角色速度属性决定，上限 4×1.7）
- 可多段命中：根据 `(Speed + 武功类型属性) / 180` 的分数随机判定 1~3 段
- 每段效果依次沿突进路径偏移，`Frame` 递增 3（视觉错开）

**命中效果：**
- 伤害除以 **1.5**（减伤）
- 施加 **冰冻效果** 5 帧（`applyFrozen(r, 5)`）
- 标准击退（强度 2.0，上限 3.0）

---

### OperationType 4 — 暗器

**创建规则：**
- 自动瞄准最远敌人
- 速度 10（快速飞行）
- `TotalFrame = 100`
- 不贯穿（`Through = 0`）
- 使用暗器自带的效果贴图

**命中效果：** 走暗器伤害公式 `calHiddenWeaponHurt()` 而非武功伤害公式

---

## 三、通用命中机制

1. **判定条件**：与敌人距离 ≤ `TILE_W * 2`，且该敌人未处于受伤无敌帧、未死亡、未被此效果命中过
2. **角度修正**：攻击方向与防御者面朝方向的夹角影响伤害
   - 正面（< 45°）：1.0 倍
   - 侧面（45°~135°）：1.2 倍
   - 背面（> 135°）：1.5 倍
3. **击退**：命中后对防御者施加远离攻击者方向的速度，OperationType 2 的击退较弱
4. **无敌帧**：命中后设 `HurtFrame = 1`，在无敌帧内不会被再次命中
5. **保底伤害**：若计算后伤害 ≤ 0，则随机给 1~4 点伤害
