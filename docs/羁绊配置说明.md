# 羁绊配置说明

本文档介绍如何通过修改配置文件来调整自走棋羁绊（combo）系统的数值。

## 配置文件位置

```
game/config/chess_combos.yaml
```

游戏启动后会自动读取此文件。如果文件格式有误，控制台会输出具体的错误信息。

## 基本结构

```yaml
羁绊:
  - 名称: 江南七怪
    成员: [柯鎮惡, 朱聰, 韓寶駒, 南希仁, 張阿生, 全金發, 韓小瑩]
    阈值:
      - 人数: 3
        名称: 七怪合力
        效果:
          - 类型: 防御加成
            数值: 10
```

每个羁绊包含：

| 字段 | 说明 |
|------|------|
| 名称 | 羁绊的显示名称 |
| 成员 | 属于该羁绊的角色名字列表（必须使用繁体中文，与游戏数据库一致） |
| 反向羁绊 | 可选，设为 `true` 表示人数越少效果越强（如「独行」） |
| 阈值 | 激活条件列表，按人数从少到多排列 |

每个阈值包含：

| 字段 | 说明 |
|------|------|
| 人数 | 需要上场的该羁绊成员数量 |
| 名称 | 该档位的显示名称 |
| 效果 | 该档位提供的效果列表 |

## 效果配置

每个效果的基本格式：

```yaml
- 类型: 攻击百分比
  数值: 15
```

部分效果有额外参数：

```yaml
# 附加参数：用于持续时间、间隔等
- 类型: 中毒伤害
  数值: 2          # 每帧伤害百分比
  附加参数: 90     # 持续90帧

# 触发条件：效果仅在特定条件下生效
- 类型: 攻击百分比
  数值: 30
  触发: 低血量
  触发参数: 40     # 血量低于40%时生效

# 友方低血狂暴：友方血量低时触发临时增益
- 类型: 攻击百分比
  数值: 50
  触发: 友方低血狂暴
  触发参数: 30     # 血量低于30%时触发
  持续帧数: 120    # 持续120帧
```

## 所有效果类型

### 属性加成（战前生效）

| 类型 | 说明 | 数值含义 |
|------|------|----------|
| 生命加成 | 固定增加生命值 | 加成数值 |
| 攻击加成 | 固定增加攻击力 | 加成数值 |
| 防御加成 | 固定增加防御力 | 加成数值 |
| 速度加成 | 固定增加速度 | 加成数值 |
| 生命百分比 | 按百分比增加生命值 | 百分比 |

| 防御百分比 | 按百分比增加防御力 | 百分比 |
| 防御削减 | 按百分比降低防御力 | 百分比 |

## 常见调整示例

**调整数值**：直接修改 `数值` 字段即可。例如将江南七怪3人档的防御加成从10改为20：

```yaml
- 类型: 防御加成
  数值: 20
```

**调整激活人数**：修改 `人数` 字段。例如将五绝从需要4人改为3人激活最高档：

```yaml
- 人数: 3
  名称: 天下五绝
```

**添加新羁绊**：在 `羁绊:` 下添加新条目，成员名字必须与上方角色名完全一致。

**添加新效果**：在对应阈值的 `效果:` 下添加新条目即可。
