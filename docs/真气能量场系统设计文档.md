# 真气能量场系统 - 设计文档

> **版本**：v2.0（完整物理模型）
> **创建日期**：2026-03-02
> **设计理念**：真气 = 沿经脉流动、用 Perlin 做飘逸效果、可外放可压缩、带属性的能量场

---

## 一、核心设计理念

### 1.1 什么是真气？

**真气不是纯粹的视觉特效，而是一个有物理行为的能量系统。**

```
真气 = 能量微粒 + 密度场 + 流动脚本
```

- **能量微粒**（QiParticle）：最小的能量单元
- **密度场**（QiField）：空间中的真气分布
- **流动脚本**（QiFlowScript）：功法如何操控真气

### 1.2 核心公式

```cpp
真气密度衰减：density(d) = density₀ × e^(-d/λ)
真气扰动：turbulence_scale = 1 / (1 + speed × 0.05)
五行相克：damage_multiplier = 1.5 (when counter)
压缩倍率：explosive = 10×, compressed = 3×, normal = 1×, dispersed = 0.3×
```

## 三、真气状态（内外两态）

### 3.1 状态枚举

| 状态 | 说明 | 粒子行为 | 典型功法 |
|------|------|---------|---------|

```
Internal → External  (内功外放)
External → Shield    (收回护体)
Shield → Healing     (传功疗伤)
```

## 五、真气压缩度

### 5.1 压缩级别

| 压缩度 | 密度倍率 | 粒子表现 | 应用场景 |
|--------|---------|---------|---------|
| **Dispersed** | 0.3× | 松散、范围大、威力低 | 群体攻击、毒雾 |
| **Normal** | 1.0× | 正常 | 常规攻击 |
| **Compressed** | 3.0× | 密集、范围小、威力高 | 精准打击 |
| **Explosive** | 10.0× | 极度压缩、瞬间爆发 | 大招、爆发技 |

### 5.2 压缩过程

以**降龙十八掌**为例：

```
阶段1（0-0.2秒）：压缩 (Compressed)
    ├─ 粒子向中心聚集
    ├─ 密度 ↑↑ (× 2 per second)
    └─ 温度 ↑↑ (× 3 per second)

阶段2（0.2秒后）：爆发 (Explosive)
    ├─ 粒子沿方向高速释放 (3× speed)
    ├─ 密度 ↓ (× 0.5 per second)
    └─ 扰动 ↓↓ (turbulence = 0.02, 笔直推进)
```

## 七、物理模拟

### 7.1 粒子更新（每帧）

```cpp
1. 更新寿命：lifetime -= dt
2. 更新速度：velocity += acceleration × dt
3. 更新位置：pos += velocity × dt
4. 应用湍流：
   turb_x = perlinNoise(pos.x, seed) × turbulence × 10
   pos += turb × dt
5. 淘汰死亡粒子：lifetime <= 0
```

### 7.2 真气场采样

```cpp
// 密度场（高斯分布）
density(pos) = field.density × e^(-normalized_dist² × 2) × compression_multiplier

// 速度场（径向流动）
velocity(pos) = flow_direction × flow_speed × (1 - dist/radius)
```

### 7.3 Perlin 噪声湍流

```cpp
turbulence = perlinNoise(pos.x × 0.05, pos.y × 0.05, seed) × strength

// 扰动强度由 turbulence 参数控制
turbulence_scale = 1 / (1 + speed × 0.05)  // 速度越快 → 扰动越小
```

## 八、渲染策略

### 8.1 粒子渲染

```cpp
screen_x = pos.x - camera_offset.x
screen_y = pos.y / 2.0 - camera_offset.y  // 2.5D 压缩

radius = density × 2.0
alpha = density × life_ratio × 150

for dy in [-radius, radius]:
    for dx in [-radius, radius]:
        if dx² + dy² <= radius²:
            renderPoint(screen_x + dx, screen_y + dy, color, alpha)
```

### 8.2 颜色计算

```cpp
// 基础颜色（根据属性）
base_color = getAttribute Color(attribute)

// 温度修正
if temperature > 1.0:
    r *= temperature, g /= temperature, b /= temperature  // 偏红
else if temperature < 0.8:
    r *= temperature, g *= temperature, b /= temperature  // 偏蓝
```

## 附录A：公式汇总

### A.1 衰减定律

```math
density(d) = density₀ × e^(-d/λ_d)
size(d) = size₀ × e^(-d/λ_s)
brightness(d) = brightness₀ × e^(-d/λ_b)
```

- **λ_d**：密度衰减长度（50px）
- **λ_s**：大小衰减长度（80px）
- **λ_b**：亮度衰减长度（50px）

### A.2 扰动公式

```math
turbulence_scale(v) = 1 / (1 + v × k)
```

- **v**：速度
- **k**：扰动系数（0.05）

### A.3 压缩倍率

```math
multiplier(c) = {
    0.3,  if c == Dispersed
    1.0,  if c == Normal
    3.0,  if c == Compressed
    10.0, if c == Explosive
}
```
