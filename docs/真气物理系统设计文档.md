## 一、系统概述

真气物理系统是一个**武侠风格粒子特效系统**，用于为战斗中的武功攻击渲染逼真的真气效果。系统采用**形状与属性分离**的架构，支持多种武功类型的可视化表现。

### 核心设计理念

1. **形状驱动**：不同武功类型有不同的真气形状（剑气、刀气、拳劲、掌风）
2. **属性叠加**：真气属性（火、冰、雷等）作为颜色和特效的修饰
3. **物理衰减**：真气随距离指数衰减，符合武侠"气劲渐散"的设定
4. **自动推断**：根据武功类型和名称自动选择合适的视觉效果

## 三、真气衰减定律

### 3.1 亮度衰减（指数衰减）

```cpp
brightness = initial_brightness × e^(-distance / brightness_decay_coeff)
```

- **distance**：粒子距离发射源的距离
- **brightness_decay_coeff**：衰减系数（默认 50.0）
- **效果**：真气越远越暗，最终消散

### 3.2 大小衰减

```cpp
size = initial_size × e^(-distance / size_decay_coeff)
```

- **size_decay_coeff**：大小衰减系数（默认 80.0）
- **效果**：真气越远越小

### 3.3 寿命衰减

```cpp
alpha = base_alpha × (lifetime / max_lifetime) × distance_decay
```

- **lifetime**：剩余寿命（秒）
- **max_lifetime**：最大寿命（默认 0.5秒）
- **效果**：随时间淡出

### 3.4 距离衰减率

```cpp
外放距离 = 基础距离 × (1 - 距离衰减率 × 移动步数)
```

- **距离衰减率**：默认 0.02（每步衰减2%）
- **效果**：控制真气的最大传播距离

## 六、物理模拟

### 6.1 粒子更新（每帧）

```cpp
1. 更新寿命：lifetime -= delta_time
2. 更新位置：pos += velocity * delta_time
3. 应用湍流：pos += perlin_noise(pos) * turbulence_strength
4. 应用重力：velocity.y += gravity * delta_time
5. 更新旋转：rotation += delta_time * 0.5（剑气/刀气）
6. 裁剪淘汰：lifetime <= 0 或 brightness < 0.05
```

### 6.2 Perlin 噪声湍流

```cpp
turbulence_x = perlinNoise(pos.x * 0.05 + seed) * strength
turbulence_y = perlinNoise(pos.y * 0.05 + seed) * strength
```

- **作用**：产生自然的气流摆动
- **强度**：默认 2.0px

## 七、性能优化

### 7.1 粒子池

```cpp
static constexpr size_t MAX_PARTICLES = 2000;
```

- **策略**：预分配内存，避免频繁分配
- **淘汰**：超过上限时停止生成

### 7.2 渲染裁剪

```cpp
if (brightness < 0.05f || size < 1.0f) return;
```

- **作用**：跳过几乎不可见的粒子
- **性能提升**：约30%

### 7.3 距离裁剪

```cpp
float distance = particle.pos.norm();
```

- **作用**：远离源点的粒子优先淘汰

## 附录：公式汇总

### A.1 衰减公式

```math
brightness(d) = brightness₀ × e^(-d / λ_b)
size(d) = size₀ × e^(-d / λ_s)
distance(n) = distance₀ × (1 - α × n)
```

- **λ_b**：亮度衰减长度（50px）
- **λ_s**：大小衰减长度（80px）
- **α**：距离衰减率（0.02）
- **n**：移动步数

### A.2 Perlin 噪声（1D）

```cpp
noise(x) = interpolate(
    hash(floor(x)),
    hash(floor(x) + 1),
    smoothstep(frac(x))
)

smoothstep(t) = 3t² - 2t³
```
