# 真气特效渲染器 - 使用文档

> **版本**：v3.0（合并版）
> **合并内容**：QiParticleSystem（形状）+ QiEnergySystem（功法）
> **当前用途**：武功特效视觉显示
> **物理公式**：保留在 `QiEffect::PhysicsLaws` 命名空间，供以后扩展

---

## 一、快速使用

### 1.1 基本用法

```cpp
#include "QiEffectRenderer.h"

// 创建渲染器
QiEffect::Renderer qi_renderer;

// 每帧渲染（在 BattleSceneHades::draw() 中调用）
for (auto& attack_effect : attack_effects_)
{
    qi_renderer.render(renderer, attack_effect, current_frame_);
}
```

### 1.2 调整全局参数

```cpp
// 调整全局亮度
qi_renderer.setGlobalBrightness(0.8f);  // 80% 亮度

// 禁用粒子特效（提升性能）
qi_renderer.setEnableParticles(false);
```

## 四、视觉参数详解

### 4.1 形状（Shape）

```cpp
enum class Shape {
    FistWave,      // 拳劲：圆形冲击波
    SwordQi,       // 剑气：细长气刃
    BladeQi,       // 刀气：宽厚气刃
    PalmWind,      // 掌风：扇形气浪
    SpecialAura,   // 特殊：飘带
};
```

### 4.2 属性（Attribute）

```cpp
enum class Attribute {
    None,   // 白色 (220,220,220)
    Metal,  // 白金 (220,220,240) ← 剑法
    Wood,   // 绿色 (120,220,120)
    Water,  // 蓝色 (100,180,255)
    Fire,   // 红色 (255,120,80)  ← 刀法
    Earth,  // 黄色 (200,160,100)
    Yang,   // 金黄 (255,220,100) ← 拳法
    Yin,    // 紫黑 (160,100,200) ← 特殊
};
```

**温度影响**：
- 高温（>1.0）→ 偏红色
- 低温（<0.9）→ 偏蓝色

### 4.3 压缩度（Compression）

```cpp
enum class Compression {
    Dispersed,  // 松散：范围×0.3，用于群体攻击
    Normal,     // 正常：范围×1.0
    Compressed, // 压缩：范围×3.0，精准打击
    Explosive,  // 爆发：范围×10.0，大招
};
```

**当前使用**：
- 拳法：Explosive（爆发型，降龙十八掌）
- 剑法：Compressed（压缩，精准）
- 刀法：Normal（正常）
- 掌风：Dispersed（松散，范围大）

### 4.4 扰动度（Turbulence）

```cpp
turbulence = 0.01  // 极低扰动 → 笔直锐利（剑气）
turbulence = 0.03  // 低扰动 → 较直（刀气）
turbulence = 0.05  // 中低扰动 → 稳定（拳劲）
turbulence = 0.08  // 中等扰动 → 飘逸（飘带）
turbulence = 0.10  // 高扰动 → 飘荡（掌风）
```

## 五、物理公式（保留接口）

**当前不影响游戏逻辑，仅用于视觉计算。将来可扩展为伤害计算。**

### 5.1 密度衰减

```cpp
QiEffect::PhysicsLaws::densityDecay(initial_density, distance, decay_length);
// density(d) = density₀ × e^(-d/λ)
```

### 5.2 压缩倍率

```cpp
QiEffect::PhysicsLaws::compressionMultiplier(compression);
// Explosive = 10×, Compressed = 3×, Normal = 1×, Dispersed = 0.3×
```

### 5.3 扰动缩放

```cpp
QiEffect::PhysicsLaws::turbulenceScale(speed);
// scale = 1 / (1 + speed × 0.05)
// 速度越快 → 扰动越小 → 越笔直
```

### 5.4 五行相克

```cpp
QiEffect::PhysicsLaws::attributeCounter(attacker_attr, defender_attr);
// 金克木，木克土，土克水，水克火，火克金
// 返回：1.5× (相克), 1.3× (阴阳克), 1.0× (无克制)
```

**用途**：将来可用于伤害计算。

## 八、性能优化

### 8.1 当前优化

- ✅ 透明度裁剪：alpha < 5 不渲染
- ✅ 命中后消失：`if (!ae.Defender.empty()) return`
- ✅ 粒子数量控制：particle_count / 5

### 8.2 进一步优化

如果性能还有问题，可调整：

```cpp
// 减少拖尾长度
params.trail_length = 30.0f;  // 默认 40-50

// 减少扇形密度
for (float angle = ...; angle += 0.1f)  // 默认 0.05
```

## 十一、常见问题

### Q1: 如何调整剑气的宽度？

```cpp
// 在 inferParams() 中修改剑法分支
case 2:  // 剑法
    params.width_max = 20.0f;  // 默认 16.0f
    params.width_min = 3.0f;   // 默认 2.0f
```

### Q2: 如何让拳法也有拖尾而非冲击波？

```cpp
// 修改拳法形状
case 1:  // 拳法
    params.shape = Shape::SpecialAura;  // 改为飘带
```

### Q3: 如何统一所有武功的颜色？

```cpp
// 在 inferParams() 中，所有分支都设置
params.attribute = Attribute::Yang;  // 统一金黄色
```
