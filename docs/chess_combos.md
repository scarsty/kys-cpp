# 自走棋羁绊系统设计

## 概述

每个羁绊由若干棋子组成，达到指定数量阈值时激活对应效果。
效果仅对该羁绊内的成员生效。部分棋子可同时属于多个羁绊，形成选秀博弈。

### 23. 独行

绝顶高手，不屑群战。反羁绊机制：场上独行标签的棋子越少，效果越强。

| 成员 | 费用 |
|------|------|

| 裘千仞 | 3 |
| 段延庆 | 3 |
| 萧远山 | 3 |
| 欧阳锋 | 4 |
| 扫地老僧 | 5 |
| 王重阳 | 5 |

| 条件 | 效果名 | 效果 |
|------|--------|------|
| 场上仅1名独行成员 | 绝世独立 | 该棋子攻击+25%，防御+20%，速度+15，受到伤害-15%(减伤) |
| 场上有2名独行成员 | 惺惺相惜 | 独行成员攻击+12%，防御+10%，速度+8 |
| 场上有3名及以上 | 泯然众人 | 独行成员仅获得攻击+5% |

设计思路：反羁绊——人越少越强。扫地老僧、王重阳等绝顶高手独自上场时获得巨大增益，鼓励玩家只放一个独行棋子。与传统"凑人数"的羁绊形成策略对抗。

## 速度 vs 冷却缩减 说明

- **速度(Speed)**：影响移动速度(`Speed/30.0`，上限4.0)、闪身资格(`Speed>=60`)、闪身攻击次数
- **冷却缩减(Cooldown Reduction)**：直接减少技能释放后的等待帧数，使攻击更频繁
- 两者独立，速度让棋子跑得快/能闪避，冷却缩减让棋子打得快
