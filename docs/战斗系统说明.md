# 战斗系统说明

本文档介绍即时战斗场景的核心系统与交互逻辑。

## 帧驱动

所有战斗系统基于帧计数运行。每帧递减各类计时器（僵直、震动、冷却、无敌等），到0时效果结束。

## 操作类型

棋子的攻击方式分为5种操作类型：

| 名称 | 说明 | 施法帧 | 基础冷却 |
|------|------|--------|----------|
| 轻击 | 单体弹射攻击，有连击计数器，第3次命中或大招时强化为2倍伤害 | 20 | 75 |
| 重击 | 扇形范围弹射，大招时弹射数翻倍 | 25 | 180 |
| 远程 | 直线穿透弹射，附带侧翼弹射 | 15 | 100 |
| 闪避 | 位移闪避，无伤害 | 8 | 15 |

冷却公式：最低冷却与（基础冷却 - 熟练度加成）取较大值，再乘以速度修正（速度越高冷却越短，最多缩短约33%），最后乘以冷却缩减修正（来自羁绊，详见羁绊配置说明）。

## 攻击范围类型

定义在武功上，决定技能的覆盖形状，并自动映射到操作类型：

| 名称 | 覆盖 | 映射操作类型 |
|------|------|-------------|
| 点 | 单体 | 轻击 |
| 线 | 直线穿透 | 远程 |
| 十字 | 十字穿透 | 远程 |
| 面 | 全范围AOE | 重击 |

## 伤害类型

| 名称 | 说明 |
|------|------|
| 普通 | 造成生命伤害 |
| 吸取内力 | 消耗对方内力而非生命 |

## 武功类型

武功分为拳掌、剑法、刀法、奇门四类。棋子对应类型的熟练度越高，使用该类武功的冷却越短。

| 类型 | 说明 | 代表武功举例 |
|------|------|-------------|
| 拳掌 | 以拳法、掌法为主 | 降龍十八掌（郭靖、喬峰）、黯然銷魂掌（楊過）、蛤蟆功（歐陽鋒）、龍象般若功（金輪法王）、易筋神功（掃地老僧）、先天功（王重陽） |
| 剑法 | 以剑术为主 | 六脈神劍（段譽）、玉女素心劍（小龍女）、玄鐵劍法（楊過）、金蛇劍法（袁承志）、斗轉星移（慕容復） |
| 刀法 | 以刀术为主 | 南山刀法等 |
| 奇门 | 特殊兵器或异类武功 | 打狗棒法（黃蓉、洪七公） |

同一个角色可能同时拥有不同类型的武功。例如楊過既有剑法（玄鐵劍法，十字范围）又有拳掌（黯然銷魂掌，单体点范围）；洪七公既有奇门（打狗棒法，十字范围）又有拳掌（降龍十八掌，单体点范围）。

## 僵直

僵直是控制机制，使棋子完全无法行动（不能移动、攻击、使用道具）。

- 僵直有一个帧计数器，每帧递减，到0时解除
- 场景级僵直会暂停整个战斗循环并触发手柄震动，用于打击停顿效果

羁绊可提供僵直抗性（详见羁绊配置说明）：
- 僵直抗性：按百分比减少僵直持续时间（如逍遥6人的僵直抗性100即完全免疫）
- 护盾僵直抗性：护盾存在时的额外僵直抗性（如全真教的护盾僵直抗性100）
- 免控帧数：僵直持续时间硬上限（如五绝4人的免控帧数15）

## 震动

纯视觉效果。震动期间棋子贴图随机偏移，无实际战斗影响。

## AI技能选择

AI决策流程：

1. **选择武功**：
   - 内力满 → 大招模式，选择攻击力**最高**的武功
   - 否则 → 普通模式，选择攻击力**最低**的武功（节省资源）

2. **移动到攻击范围**：
   - 寻找最近敌人
   - 根据武功的攻击范围类型计算所需距离（点=近战、线/十字=弹射距离、面=中距离）
   - 移动途中有25%概率使用闪避

3. **发动攻击**：
   - 75%概率攻击，25%概率闪避

## 大招系统

触发条件：内力值满。

效果：
- 选择最强武功
- 显示金色文字特效
- 伤害翻倍
- 额外生成±30°追踪弹射
- 释放后内力清零、连击计数重置

## 连击计数

仅对轻击生效：
- 每次轻击连击计数+1
- 第3次命中或大招时触发强化攻击（2倍伤害）
- 超过一定时间未攻击或触发强化后重置

## 伤害计算

基础公式：攻击力 = 角色攻击 + 武功威力 / 3

修正因子：
- 减去弱化值
- 乘以强化倍率（大招时为2）
- 距离衰减：弹射飞行越远伤害越低，最远处伤害减半

属性加成来自星级加成（见平衡配置说明）和羁绊效果（见羁绊配置说明）。

## 弹射系统

弹射速度固定为每帧5像素。弹射飞行帧数 = 15 + 选择距离 × 5。弹射最远距离 = 生成偏移 + 速度 × 帧数。

## 战斗流程总结

1. **AI决策** → 选择武功（普通选最弱，大招选最强）
2. **移动** → 向目标移动至攻击范围内
3. **施法** → 前摇动画（8~25帧）
4. **攻击** → 生成弹射/效果
5. **冷却** → 后摇（15~180帧，受速度和羁绊影响）
6. **结算** → 命中检测、伤害计算、状态效果
7. **羁绊处理** → 更新增益/减益状态
